# 漫画-小说在线阅读平台 - 模块划分清单
基于 Python Flask + Vue3 技术栈

## 一、核心功能模块（高优先级）

### 1. 用户管理模块
**目标：安全认证、身份管理、多端一致性**

| 功能 | 描述 | 优先级 | 标记 |
|------|------|--------|------|
| 用户注册/登录 | 支持用户名+密码登录 | 高 | ✓ |
| JWT Token身份验证 | API认证，Token刷新机制 | 高 | ✓ |
| 用户资料管理 | 编辑用户名、密码、昵称 | 高 | ✓ |
| 权限管理 | 读者/管理员角色，支持切换 | 高 | ✓ |
| 密码找回 | 邮箱/短信验证码重置 | 高 | + |
| 封禁/解封用户 | 管理员后台操作 | 高 | + |

### 2. 内容管理模块
**目标：高效上传、内容优化、版本可控**

| 功能 | 描述 | 优先级 | 标记 |
|------|------|--------|------|
| 书籍上传/编辑 | 富文本编辑器，存数据库/文件 | 高 | ✓ |
| 漫画上传 | 支持JPG/PNG等图片格式 | 高 | ✓ |
| 批量上传功能 | 多文件同时上传 | 高 | ✓ |
| 多级分类体系/标签系统 | 如：玄幻>仙侠；标签：热血、穿越 | 高 | ✓ |
| 作者/出版社管理 | 添加/编辑作者、出版社信息 | 高 | ✓ |
| 封面图上传与自动裁剪 | 统一尺寸，提升展示 | 高 | + |
| 文件格式校验与转码 | EPUB→HTML流式，CBZ→分页图 | 高 | + |
| 存储优化 | CDN + 分片存储 | 高 | + |

### 3. 阅读功能模块
**目标：流畅体验、个性化设置、进度同步**

| 功能 | 描述 | 优先级 | 标记 |
|------|------|--------|------|
| 在线阅读器 | 支持书籍/漫画浏览 | 高 | ✓ |
| 漫画分页阅读 | 单页/连续滚动 | 高 | ✓ |
| 书籍流阅读 | 连续文本流，支持翻页/滚动 | 高 | ✓ |
| 阅读进度保存 | 自动记录最后位置 | 高 | ✓ |
| 阅读设置 | 字体、背景、亮度、夜间模式、阅读方向 | 高 | ✓ |
| 目录导航（书籍） | 章节列表跳转 | 高 | + |
| 划词注释/高亮 | 添加笔记、高亮文本 | 高 | + |
